# Test Strategy — Project Config (hangar-bridge)

## Test Commands
- All packages: `pnpm -r test:ci`   (vitest run + coverage where configured)
- Watch: `pnpm -r test`
- Single package: `pnpm -F @hangar-bridge/<pkg> exec vitest run`
- Single file: `pnpm -F @hangar-bridge/shared exec vitest run channel`
- Single test name: `pnpm -F @hangar-bridge/shared exec vitest run -t "round-trip"`
- Typecheck: `pnpm -r typecheck`
- Lint/format: none configured (no `lint` script in any package).

## Test Pyramid
- Unit (vitest, per package) is primary. `packages/e2e/` holds the cross-package / loopback integration tests.
- Property-based tests use `fast-check` (e.g. the 500-run `<channel>` body-escaping invariant in `shared`).
